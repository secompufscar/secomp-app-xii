### Como rodar

1. Clone the repositório
    ```sh
    git clone https://github.com/secompufscar/secomp-app-xii.git
    ```

2. Acesse a pasta clonada

    ```sh
    cd ./secomp-app-xii
    ```

3. Instale as dependências

    ```sh
    npm install
    ```

4. Execute o programa

    ```sh
    npm start
    ```
